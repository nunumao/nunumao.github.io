Kotlin (三) | 函数

KOTLIN 2025-02-10 15 分钟阅读 0

函数 (Functions)

函数是程序的基本构建块，它允许我们将代码组织成可重用的模块。 函数可以接受输入 (参数)，执行特定的任务，并返回输出 (返回值)。 在 Kotlin 中，函数是头等公民，这意味着函数可以像变量一样被赋值、传递和返回。

函数的定义和调用

在 Kotlin 中，我们使用 fun 关键字来定义函数。

语法:

  • fun: 关键字，表示这是一个函数定义。
  • 函数名: 函数的名称，遵循标识符命名规则，通常使用驼峰命名法 (例如 calculateSum, getUserName)。
  • 参数列表: 可选的，用括号 () 包裹，包含函数的参数。 参数格式为 参数名: 参数类型, 参数名: 参数类型, ...。 如果没有参数，括号内为空 ()。
  • 返回值类型: 指定函数返回值的类型。 如果函数没有返回值 (类似于 Java 中的 void)，则返回值类型为 Unit (可以省略)。 Kotlin 有返回值类型推断，在某些情况下可以省略返回值类型声明。
  • 函数体: 用花括号 {} 包裹的代码块，包含函数要执行的语句。
  • return 返回值: 可选的，如果函数有返回值，使用 return 关键字返回结果。 如果没有返回值，可以省略 return 语句 (或者使用 return Unit)。

函数参数 (Parameters) 和返回值 (Return Values)

  • 函数参数: 函数接收的输入值。 每个参数都需要指定名称和类型。

  • 返回值类型: 函数执行后返回的结果的类型。 如果函数没有返回值，使用 Unit 类型 (可以省略)。

  • Unit 返回类型: 类似于 Java 的 void。 表示函数不返回任何有意义的值。 如果函数没有显式声明返回值类型，并且没有 return 语句，则返回值类型默认为 Unit。

函数类型 (Function Types) 和 Lambda 表达式 (Lambda Expressions) - 初步介绍

在 Kotlin 中，函数可以作为值来使用，就像 Int, String 等类型一样。 函数类型 用于表示函数的类型。

函数类型的语法: (参数类型1, 参数类型2, ...) -> 返回值类型

例如：

  • (Int, Int) -> Int: 表示接受两个 Int 参数，并返回 Int 类型的函数。
  • (String) -> Unit: 表示接受一个 String 参数，没有返回值的函数 (返回 Unit)。
  • () -> String: 表示没有参数，返回 String 类型的函数。

Lambda 表达式 是一种匿名函数 (没有函数名的函数) 的简洁表示方式。 Lambda 表达式可以赋值给函数类型的变量，也可以作为参数传递给其他函数。

Lambda 表达式的语法: { 参数列表 -> 函数体 }
