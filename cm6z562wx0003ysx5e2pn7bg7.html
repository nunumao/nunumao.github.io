控制流 (Control Flow)

控制流语句允许我们根据条件或重复执行特定的代码块。 本章我们将学习 Kotlin 中常用的控制流语句。

条件语句 (Conditional Statements): if, else if, else

if 语句用于根据条件执行不同的代码块。 Kotlin 的 if 语句与 Java 和其他语言类似，但有一些 Kotlin 特有的特性。

语法:

  • 条件: 一个布尔表达式 (结果为 true 或 false)。
  • 代码块: 用花括号 {} 包裹的一段代码。 如果代码块只包含一行语句，花括号可以省略 (但不推荐为了代码可读性，最好总是加上花括号)。
  • else if: 可以有零个或多个 else if 分支。
  • else: 最多只有一个 else 分支，并且必须放在所有 else if 分支之后。

在 Kotlin 中，if 不仅是语句，还可以作为表达式，这意味着 if 可以返回一个值。 这在很多情况下非常方便。

当 if 作为表达式时，if 和 else 分支都必须有返回值 (除非返回值类型是 Unit，即 void)。 最后一个表达式的值就是 if 表达式的返回值。

when 表达式 (When Expression)

when 表达式是 Kotlin 中更强大、更灵活的条件语句，类似于 Java 和 C++ 中的 switch 语句，但功能更强大。

语法:

  • 表达式: 可以是任何类型的表达式。
  • 值: 要匹配的值，可以是常量、变量、表达式等。 可以使用逗号 , 分隔多个值。
  • 范围: 可以使用 .. 运算符表示范围 (例如 1..10)。
  • 类型: 可以使用 is 关键字检查类型。
  • else: 类似于 switch 语句的 default 分支，当所有 case 都不匹配时执行。 当 when 作为表达式使用时，else 分支通常是必需的，除非编译器能推断出所有可能的情况都已覆盖。

when 也可以不带参数，这时每个分支的条件都是一个独立的布尔表达式。 类似于 if-else if-else 结构。

总结: when 表达式比 if-else if-else 结构更简洁、更易读，功能更强大，是 Kotlin 中推荐使用的条件语句。

循环 (Loops): for, while, do-while

循环语句用于重复执行一段代码块。 Kotlin 提供了 for, while, do-while 三种循环。

  • for 循环: 用于遍历集合 (例如列表、数组、范围等) 或执行固定次数的循环。

  • while 循环: 当条件为 true 时，重复执行代码块。 先判断条件，后执行循环体。

  • do-while 循环: 先执行一次代码块，然后判断条件，如果条件为 true，则继续重复执行。 至少执行一次循环体。

break 和 continue

  • break: 用于立即终止循环，跳出循环体。
  • continue: 用于跳过当前循环迭代的剩余代码，直接进入下一次迭代。
