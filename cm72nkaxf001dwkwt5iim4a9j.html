机器学习 (一) | 概述

PYTHON 2025-02-04 15 分钟阅读 0

什么是机器学习？

想象一下，你想教你的电脑识别猫和狗的照片。 你不会写一堆“如果耳朵是尖的，那就是狗”这样的规则，因为猫也可能有尖耳朵。 机器学习就是让电脑自己从大量猫和狗的照片中学习，找到区分它们的模式。

Sklearn 是什么？

Sklearn (全称 Scikit-learn) 就像一个装满了各种机器学习工具的工具箱。 它已经帮你写好了很多常用的机器学习“算法”，你只需要把你的数据“喂”给它，告诉它你想做什么，它就能帮你完成任务。 这就像你不用自己发明螺丝刀，可以直接从工具箱里拿来用一样方便。

Sklearn 的核心概念

  • 数据 (Data): 这是机器学习的燃料。 通常，数据会 organized 成一个表格，就像 Excel 表格一样。

    • 行 (Row): 每一行代表一个 “样本” 或者 “数据点”。 比如，一张猫的照片算一行。
    • 列 (Column): 每一列代表一个 “特征” 或者 “属性”。 比如，照片的“颜色”、“大小”、“耳朵形状”等等。

  • 特征 (Features): 描述数据的属性，就像猫照片的“颜色”、“大小”。 机器学习算法会根据这些特征来学习。

  • 标签 (Labels) / 目标变量 (Target Variable): 这是你想让机器学习预测的东西。 比如，对于猫和狗的图片，标签就是“猫”或者“狗”。 对于预测房价，标签就是“房价”。

  • 模型 (Model): 这是机器学习算法学习到的“规则”或者“模式”。 就像你大脑里区分猫和狗的 “认知模式”。

  • 训练 (Training): 这是让模型学习数据的过程。 你把带有标签的数据 “喂” 给算法，算法会调整自己内部的参数，以便更好地预测新的数据。

  • 预测 (Prediction): 训练好模型后，你就可以用它来预测新的、没有标签的数据了。 比如，给模型一张新的照片，让它预测是猫还是狗。

Sklearn 里的“角色”

Sklearn 里面有很多“角色”，我们称之为“算法”。 不同的算法擅长解决不同的问题。 下面我们介绍一些常见的“角色”，就像认识新朋友一样：

1. 分类器 (Classifier): 判断类别

分类器的任务是把数据分到不同的类别里。 就像给水果分类，苹果放一堆，香蕉放一堆。

  • 决策树 (Decision Tree): 像一个流程图一样做决定。 它会根据不同的特征一层层地判断，最终到达一个类别。

  • 逻辑回归 (Logistic Regression): 虽然名字里有“回归”，但它其实是个分类器。 它会根据特征计算出某个样本属于某个类别的概率。 就像预测一个学生及格还是不及格。

  • 支持向量机 (Support Vector Machine, SVM): 它会找到一个 “最佳的分割线” (或者更高维度的 “超平面”) 来分隔不同的类别。 就像在两堆球中间画一条线，把它们分开。

  • K-近邻 (K-Nearest Neighbors, KNN): “近朱者赤，近墨者黑”。 它会看看一个新数据点周围最近的 K 个邻居都是什么类别，然后把这个新数据点也归到那个数量最多的类别。

  • 朴素贝叶斯 (Naive Bayes): 它利用概率来做分类。 它会假设各个特征之间是相互独立的（所以叫“朴素”），然后根据每个特征在不同类别中出现的概率来预测新数据的类别。

2. 回归器 (Regressor): 预测数值

回归器的任务是预测一个连续的数值。 就像预测房价、温度等等。

  • 线性回归 (Linear Regression): 试图找到一条直线 (或者更高维度的平面) 来最好地拟合数据点。 就像画一条线，让它尽可能地靠近所有的数据点。

  • 决策树回归 (Decision Tree Regressor): 和分类决策树类似，但是它的叶子节点预测的是一个数值，而不是一个类别。

3. 聚类器 (Clusterer): 分组

聚类器的任务是把数据分成不同的组 (称为 “簇”)，组内的成员比较相似，组间的成员比较不同。 就像把一堆没有标签的水果分成不同的堆，你可能根据颜色和大小来分组。

  • K-均值 (K-Means): 它会随机选择 K 个中心点，然后把每个数据点分配到离它最近的中心点所在的簇。 然后重新计算每个簇的中心点，重复这个过程直到簇不再变化。

4. 降维器 (Dimensionality Reducer): 简化数据

降维器的任务是减少数据的特征数量，同时尽可能保留数据的关键信息。 就像写文章时，提炼出最重要的几句话来概括全文。

  • 主成分分析 (Principal Component Analysis, PCA): 它会找到数据中最重要的几个 “方向” (称为 “主成分”)，然后把数据投影到这些方向上，从而减少特征数量。

Sklearn 的工作流程 (非常简化版)

  1. 准备数据: 把你的数据整理成 Sklearn 可以识别的格式 (通常是一个表格)。
  2. 选择模型: 根据你的任务 (分类、回归、聚类等) 选择合适的算法。
  3. 训练模型: 用你的数据 “训练” 你选择的模型。 就像教小朋友识别形状。
  4. 评估模型: 看看你的模型学得怎么样，用一些指标来衡量它的好坏。
  5. 使用模型: 用训练好的模型来预测新的数据。

总结

Sklearn 是一个非常实用的机器学习工具箱，它提供了很多常用的算法，可以帮助我们解决各种各样的问题，从识别图片到预测趋势。 理解 Sklearn 的核心概念和不同算法的作用，就能像拥有一个强大的工具箱一样，随时准备解决问题。

这只是一个非常基础的介绍，Sklearn 还有很多更深入的内容等待你去探索。 希望这个通俗易懂的解释能帮助你迈出学习机器学习的第一步！
